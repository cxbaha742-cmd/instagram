🔐 Cybersecurity Project: Contact Info Retrieval Tool (Ethical & Authorized Use Only)

I developed a Python-based script designed to retrieve contact information (email and phone number) associated with a user's own authenticated Instagram account using valid session cookies and API requests.

🔎 Key highlights of the project:

* Replicates authenticated HTTP requests using session headers and cookies
* Handles CSRF tokens and session identifiers securely
* Uses proxy tools (e.g., Burp Suite) for traffic inspection and debugging
* Parses and processes JSON responses to extract structured contact data
* Demonstrates understanding of how web authentication and internal APIs work

🛠️ Technologies used:

* Python
* requests library
* Network interception & analysis tools
* Session-based authentication concepts
* API interaction and response handling

⚠️ Disclaimer:
This tool is strictly intended for ethical and educational purposes only, and should be used exclusively on accounts owned by the user with proper authorization.

💡 The goal of this project is to deepen my understanding of web application security, session management, and API behavior in real-world environments.
